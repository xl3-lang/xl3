Why xl3

Excel stays the authoring tool.

반복적인 데이터 채우기만 작고 검증 가능한 템플릿 언어로 이동합니다.

Excel in, Excel out

템플릿과 원본 데이터가 모두 `.xlsx` 파일입니다.

Real spreadsheet templates

레이아웃, 숫자 형식, 병합 셀은 workbook 안에 남습니다.

No macros

동작은 숨겨진 스크립트가 아니라 명시적인 셀 표현식입니다.

Conformance-tested

Stage 2 OOXML 비교로 서식과 구조 보존을 검증합니다.

npm install xl3
